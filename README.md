# Order Management System (Inventory Demo)

A custom PHP MVC application for managing customer orders, featuring a modern "Professional" dashboard.

## 🚀 Features
- **MVC Architecture**: Custom-built router and core components.
- **MySQLi Integration**: Secure and efficient database interactions.
- **Premium UI**: Modern light-themed dashboard with Glassmorphism-inspired elements.
- **Interactive Dashboard**: Filter orders by status and view detailed modal popups.
- **30 Dummy Records**: Pre-seeded data for immediate testing.

## 🛠️ Technologies
- **Backend**: PHP 7.4+
- **Database**: MySQL
- **Frontend**: Vanilla JS, Modern CSS (Flexbox/Grid)
